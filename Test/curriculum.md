- 🤖 **모드 1: AI 자동 분석**
  - 의사가 자연어로 질문: "스타틴 복용군과 비복용군의 LDL 변화 비교해줘"
  - AI가 알아서: 데이터 추출 → 적절한 통계 기법 선택 → 분석 실행 → 결과+그래프
  - AI가 "왜 이 기법을 선택했는지" 근거도 설명
  - ⚠ free_text AI 분석 기능은 Day 13에서 이미 모드 1, 2에 통합 완료

    - 범위 선택: SQL 선추림(비용 절감) / 전체 분석(정밀)
    - 비식별화 적용, 비용 안내 후 진행
- 🤖 **모드 2: 의사가 단계별 선택**
  - Step 1: 데이터 선택 (환자군, 변수, 기간)
  - Step 2: 그룹 설정 (그룹 나누기, 무작위 배정 여부)
  - Step 3: 통계 기법 선택 (지원 가능한 방법 목록 제시)
    - t-test, paired t-test (두 그룹 평균 비교)
    - chi-square (범주형 비교)
    - ANOVA (세 그룹 이상 비교)
    - 회귀분석, 로지스틱 회귀
    - 생존분석 (Kaplan-Meier)
    - "AI 추천 받기" 옵션
  - Step 4: 결과 (p-value, 신뢰구간, 그래프)
  - Step 5: 내보내기
- 🔧 라이브러리: scipy.stats, statsmodels, pingouin
